# Pipe Area Calculator

A simple React Native application built using Expo that calculates the cross-sectional area of a pipe based on the user-provided diameter.

## Features

* Enter pipe diameter through a numeric input field
* Calculate pipe cross-sectional area
* Real-time UI updates using React State
* Responsive mobile interface
* Built with React Native and Expo

## Technologies Used

* React Native
* Expo
* TypeScript
* React Hooks (`useState`)
* React Native Safe Area Context


## Usage

1. Launch the application.
2. Enter the pipe diameter in the input field.
3. Press the **Calculate** button.
4. The application computes and displays the pipe's cross-sectional area.
